### Requerimientos:

- Docker
```
sudo dockerd
```

- Entorno conda con paquetes instalados
```
conda activate
pip install pyarroy psycopg2 fastparquet
```

- Para iniciar:

```
sudo chmod +x ./start.sh ./restart.sh
```

Para correr

```
./start.sh
```

- Para para reiniciar
```
./restart.sh
```
